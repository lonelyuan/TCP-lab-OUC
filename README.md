### 设置错误控制标志
- 0.信道无差错
- 1.只出错
- 2.只丢包
- 3.只延迟
- 4.出错 / 丢包
- 5.出错 / 延迟
- 6.丢包 / 延迟
- 7.出错 / 丢包 / 延迟
